# Rich Picture Studio — Quick Start Guide

Access the tool at: **https://richpicture.ronz.link**

Rich Picture Studio is a browser-based diagramming tool with a hand-drawn sketch aesthetic,
designed for creating Rich Pictures — informal diagrams that capture people, systems,
processes and the relationships between them.

---

## Canvas Navigation

| Action | How |
|--------|-----|
| Pan (move around) | Middle-mouse button drag, or two-finger trackpad drag |
| Zoom in / out | Scroll wheel, or pinch on trackpad |

---

## Drawing Tools (left panel)

Select a tool from the left-hand palette, then draw on the canvas.

| Tool | How to use |
|------|-----------|
| **Rectangle** | Click and drag to draw a box |
| **Ellipse** | Click and drag to draw an oval |
| **Freehand** | Click and drag to sketch a free-form shape |
| **Line** | Click and drag to draw a plain line between two points |
| **Arrow** | Click and drag to draw a directional arrow |
| **Bi-arrow** | Click and drag to draw an arrow pointing both ways |
| **Text** | Click anywhere on the canvas, type your text, press **Enter** to place it |

---

## Icons (right panel)

The right-hand panel contains a library of pre-drawn symbols (actors, systems,
communication channels, issues, etc.).

- **Drag** an icon from the panel and drop it onto the canvas, **or**
- **Click** an icon in the panel — a ghost preview follows your cursor.
  Click on the canvas to place it. Press **Esc** to cancel.
- **Drop an image file** directly onto the canvas to place it as a custom icon.
- **Paste** an image from the clipboard (Ctrl+V) to place it at the canvas centre.

---

## Selecting and Editing Elements

1. Click the **Select** tool (arrow icon) in the left palette.
2. **Click** any element to select it. A blue dashed outline appears.
3. **Shift-click** additional elements to add them to the selection.
4. **Click and drag** on empty canvas space to rubber-band select multiple elements.

With something selected you can:

| Action | How |
|--------|-----|
| **Move** | Drag the selected element(s) |
| **Resize** | Drag any of the four corner handles |
| **Edit text / label / description** | **Double-click** the element |
| **Delete** | Press **Delete** or **Backspace** |
| **Group** elements together | Select 2 or more, press **G** |
| **Ungroup** | Select the group, press **U** |

---

## Styling (top bar)

The controls along the top bar change depending on what is selected:

- **Nothing selected** — controls show drawing *defaults* that apply to newly
  created elements. The label on the left tells you which tool's defaults are shown.
- **Element(s) selected** — controls edit the selected element(s) directly.

| Control | What it does |
|---------|-------------|
| Stroke colour | The outline / line colour |
| Fill colour | The interior fill for shapes |
| Text colour | Colour of text / labels |
| Font | Typeface for text |
| Size | Font size |
| **B** | Bold |
| *I* | Italic |
| Background | Canvas background style (White / Cream / Lined / Dots / Grid) |

When an **arrow** is selected (or the arrow tool is active), an additional
**Label** text field appears — type here to add a caption along the arrow.

---

## Undo / Redo

- **Undo** — Ctrl+Z (or the ↩ button in the top bar)
- **Redo** — Ctrl+Y (or the ↪ button in the top bar)

---

## Saving Your Diagram

Use the **Save** button (💾) in the top bar to download your diagram as a
`.richpicture` JSON file. Use **Open** (📂) to reload a previously saved file.
**New** (📄) clears the canvas to start fresh.
